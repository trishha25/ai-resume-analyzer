# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python** and **Streamlit**. The application analyzes PDF resumes, detects technical skills, calculates a resume score, compares resumes with a job description (ATS matching), and provides recommendations for improvement.

---

## 🚀 Live Demo

👉 https://ai-resume-analyzer-gerfent5ccmlwoztzhe7nr.streamlit.app/

---

## 📌 Features

- 📄 Upload Resume in PDF format
- 🔍 Extract Resume Text
- 📊 Resume Score Calculation
- 🎯 ATS Resume Matching
- ✅ Technical Skill Detection
- ❌ Missing Keyword Detection
- 💡 Resume Improvement Suggestions
- 📋 Resume Preview

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/trishha25/ai-resume-analyzer.git
```

### Navigate to the project folder

```bash
cd ai-resume-analyzer
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
